Перед использованием программы Social Networks.py загрузить библиотеки (у меня потребовалось только загрузить vk_api) + зависимые (у меня не потребовалось):
pip install os
pip install sys
pip install vk_api
pip install urllib
pip install json
pip install getpass

Входные параметры - учетные записи клиентов - программа выбирает из файла Social Networks.py из директории с программой.
Каждая учетная запись - отдельная строка.
Поддерживаются: 
1. id,
2. псевдонимы,
3. ссылки про профили клиента

При запуске спрашивается логин VK/email и пароль. Ввести свой либо любой тестовый ("дружба" с клиентави в VK не требуется).

На выходе программы:
1. на экран ссыводятся детали из VK по всем клиентам (перечень выгружаемых полей легко расширять). Отобраны релевантные для классификации клиентов поля.
2. в файл users_detail.json выгружается JSON с детальными данными клиентов для классификации
3. в файл users_detail.txt выгружается информация в тестовом виде.
4. в папки с ID клиентов выгружается информация по соответствующему клиенту
